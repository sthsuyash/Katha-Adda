# Katha Sansar: Backend

## Description

This is the backend for the Katha Sansar project. It is a NestJS application that uses PostgreSQL as the database. It uses Prisma as the ORM.

## Installation

```bash
# install the dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev
$ docker-compose up dev-db -d


# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
